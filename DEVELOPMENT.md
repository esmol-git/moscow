# Варианты развития сайта

Идеи по доработке лендинга недвижимости.

## Деплой и письма
После любых правок в **`src/php/sendmail/`** (тема письма, текст, настройки) нужно заново собрать и залить на хостинг:
```bash
npm run build
npm run deploy
```
Иначе на сервере будет выполняться старая версия скрипта и в письмах не появятся изменения.

## Бэкенд и формы
- **Отправка заявок на сервер** — заменить `data-fls-form="dev"` на реальный endpoint (PHP/Node/сервис вроде Formspree), сохранять заявки в БД или отправлять в Telegram/почту.
- **Двойная проверка телефона** — после отправки формы отправлять SMS/звонок для подтверждения номера.

## Контент и SEO
- **Отдельные страницы объектов** — каталог с фильтрами (цена, район, тип), карточка объекта с галереей, описанием и кнопкой «Оставить заявку».
- **Блог / полезные статьи** — раздел «Как оформить сделку», «Проверка юридической чистоты» для привлечения из поиска.
- **Микроразметка** — Schema.org (LocalBusiness, Product для объектов) для богатых сниппетов в поиске.

## UX и аналитика
- **Чат или виджет** (Jivo, Carrot quest, Telegram) — быстрые ответы на вопросы без перехода в форму.
- **Карта офиса** — блок контактов с Яндекс.Картой и маркером.
- **Аналитика** — цели в Яндекс.Метрике/Google Analytics на отправку формы, клики по телефону, скролл до секции контактов.

## Личный кабинет (при масштабировании)
- **Клиентский кабинет** — история заявок, избранные объекты, статус сделки.
- **Кабинет риелтора** — добавление объектов, заявки, календарь показов.

## Дополнительно
- **Калькулятор ипотеки** — блок «Рассчитайте платёж» с полями сумма, срок, ставка.
- **Подбор по критериям** — короткая форма в hero: «Квартира / Дом, бюджет, город» → переход в каталог с предзаполненными фильтрами или модалка с формой подбора.
- **Многоязычность** — переключатель языка (RU/EN) и отдельные страницы или контент для экспатов.
